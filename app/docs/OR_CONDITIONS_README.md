# Как работают OR условия в GoFlow

## Основной принцип

Группа условий с оператором **ИЛИ (OR)** считается выполненной, если **хотя бы одно** условие в группе истинно.

## Пример

### Настройка условий
![OR условия](or-conditions-example.png)

На скриншоте показана связь с группой OR условий:
- Пройденные шаги = 0
- Есть яблоко = true

### Логика проверки

Воспроизведение пойдет по этому пути, если:
- **Либо** переменная "Пройденные шаги" равна 0
- **Либо** переменная "Есть яблоко" равна true
- **Либо** оба условия выполнены одновременно

### Когда путь НЕ будет доступен

Путь с OR условиями будет недоступен только если **ВСЕ** условия в группе ложны:
- "Пройденные шаги" ≠ 0 **И**
- "Есть яблоко" ≠ true

## Частые ошибки

### ❌ Неправильное понимание
"Путь должен быть доступен всегда, потому что у меня есть OR условия"

### ✅ Правильное понимание
"Путь доступен только если хотя бы одно из OR условий выполнено"

## Отладка

Если путь с OR условиями не выбирается:

1. **Проверьте значения переменных**
   ```javascript
   console.log('Пройденные шаги:', gameState.variables['steps'].value);
   console.log('Есть яблоко:', gameState.variables['hasApple'].value);
   ```

2. **Убедитесь, что хотя бы одно условие истинно**
   - Если `steps = 5` и `hasApple = false`, то ни одно условие не выполнено
   - Измените хотя бы одну переменную, чтобы условие стало истинным

3. **Проверьте приоритеты**
   - Если есть другие пути с выполненными условиями, они могут иметь более высокий приоритет
   - Пути с условиями имеют приоритет над путями без условий

## Примеры условий

### Пример 1: Простое OR
```
Условие 1: health > 50
Условие 2: hasPotion = true
```
Путь доступен если здоровье больше 50 **ИЛИ** есть зелье.

### Пример 2: Множественное OR
```
Условие 1: visitedShop = true
Условие 2: foundSecret = true  
Условие 3: defeatedBoss = true
```
Путь доступен если выполнено **любое** из трех условий.

### Пример 3: Комбинация с вероятностью
```
Условие 1: luck > 7
Условие 2: вероятность 30%
```
Путь доступен если удача больше 7 **ИЛИ** сработает 30% шанс.
