# Расширение компонента CustomControls

## Добавление кнопки помощи с диалоговым окном

В компонент `CustomControls` был добавлен функционал отображения справочной информации:

1. **Новая кнопка помощи**:
   - Добавлена кнопка с иконкой вопросительного знака справа от существующих элементов управления
   - Использована иконка `QuestionMarkCircledIcon` из библиотеки `@radix-ui/react-icons`
   - При нажатии на кнопку открывается диалоговое окно со справочной информацией

2. **Диалоговое окно**:
   - Реализовано с использованием компонента `Dialog` из библиотеки `@radix-ui/themes`
   - Содержит заголовок, основное содержимое и кнопку закрытия
   - Для предотвращения ошибки гидратации контент размещен внутри элемента `<div>` вместо `Dialog.Description`

3. **Отображение справочного содержимого**:
   - Контент загружается из Markdown-файла, расположенного в `/public/help/flow-help.md`
   - Имеется встроенное содержимое по умолчанию, которое отображается до загрузки файла или в случае ошибки
   - Для отображения Markdown используется библиотека `react-markdown`
   - Загрузка содержимого происходит при монтировании компонента с использованием `useEffect` и безопасной проверки на клиентскую среду

4. **Стилизация**:
   - Добавлены новые классы в CSS-модуль `CustomControls.module.scss`:
     - `.dialog_content` - стилизация контейнера для диалогового окна
     - `.markdown_container` - стилизация контейнера для Markdown содержимого
     - `.close_button` - стилизация кнопки закрытия диалога

5. **Исправленные проблемы**:
   - Решена проблема гидратации, связанная с вложением элементов `<h1>` внутрь `<p>`
   - Реализован корректный рендеринг Markdown-содержимого в React-приложении
   - Исправлена ошибка с неопределенной функцией `fetch` путем использования `window.fetch` с проверкой на клиентскую среду

## Установленные зависимости

- Добавлена библиотека `react-markdown` для отображения справочного контента в формате Markdown

## Структура файлов

- `src/components/CustomControls/CustomControls.tsx` - основной компонент
- `src/components/CustomControls/CustomControls.module.scss` - стили компонента
- `public/help/flow-help.md` - файл с содержимым справки в формате Markdown
