# Горячие клавиши GoFlow

## Инструменты
| Клавиша | Инструмент | Описание |
|---------|------------|----------|
| `Esc` | Выбор (Select) | Переключение на инструмент выбора |
| `1` | Нарративный узел (Narrative Node) | Создание нарративного узла |
| `2` | Узел выбора (Choice Node) | Создание узла выбора |
| `3` | Слой (Layer) | Создание слоя |
| `4` | Соединитель (Jumper) | Создание соединителя (временно недоступно) |
| `5` | Заметка (Note) | Создание заметки |

## Общие операции
| Клавиша | Действие | Описание |
|---------|----------|----------|
| `Пробел` | Временный режим перемещения | Удерживайте для временного переключения в режим перемещения холста |
| `Delete` / `Backspace` | Удаление | Удаляет выбранные узлы (работает только в режиме выбора) |
| `Enter` | Редактирование | Активирует режим редактирования для выбранного узла |
| `F` | Подогнать вид | Масштабирует холст так, чтобы все узлы были видны |

## Создание новых объектов
| Клавиша | Действие | Описание |
|---------|----------|----------|
| `Ctrl/Cmd + 1` | Создать нарративный узел | Добавляет нарративный узел |
| `Ctrl/Cmd + 2` | Создать узел выбора | Добавляет узел выбора (если текущий узел - выбор, добавляет выбор внизу) |
| `Ctrl/Cmd + 3` | Создать слой | Добавляет новый слой |

## Копирование и вставка
| Клавиша | Действие | Описание |
|---------|----------|----------|
| `Ctrl/Cmd + C` | Копировать | Копирует выбранные узлы |
| `Ctrl/Cmd + V` | Вставить | Вставляет скопированные узлы |

## Отмена и повтор действий
| Клавиша | Действие | Описание |
|---------|----------|----------|
| `Ctrl/Cmd + Z` | Отменить | Отменяет последнее действие |
| `Ctrl/Cmd + Shift + Z` | Повторить | Повторяет отмененное действие |

## Поиск
| Клавиша | Действие | Описание |
|---------|----------|----------|
| `Ctrl/Cmd + F` | Поиск | Открывает панель поиска |

## Блокирование браузерных горячих клавиш
В проекте реализована блокировка следующих горячих клавиш браузера, которые могут конфликтовать с редактором:

- `Ctrl/Cmd + 1-9`: блокируется переключение между вкладками
- `Ctrl/Cmd + D`: блокируется добавление закладки
- `Ctrl/Cmd + T/W/R`: блокируются операции с вкладками
- `Alt + Left/Right` и `Cmd + [/]`: блокируется навигация браузера назад/вперед
- `F6`: блокируется фокус на адресной строке
- `Ctrl/Cmd + Tab`: блокируется переключение между вкладками

> Примечание: Большинство горячих клавиш блокируются только тогда, когда фокус не находится в поле ввода текста. Операции копирования/вставки (`Ctrl/Cmd + C/V/X/A`) и форматирования текста (`Ctrl/Cmd + B/I/U`) разрешены в полях ввода.
